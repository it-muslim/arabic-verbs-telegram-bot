# terergram-arabic-verb-bot

This is a simple telegram bot for training your Arabic verbs.

# Basic usage example

```
> /play # Start a quiz
    What is Arabic for: "(He) said"? <
           1. 
           2.
           3.
           4.
> 4
> Great job!


> /play
    What is Arabic for: "(I) know"? <
           1. 
           2.
           3.
           4.
> 3
              Incorrect, the correct <
              answer is: ...
```
